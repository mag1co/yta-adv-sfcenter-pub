Requirements

YouTrack 2024.3+ — minimum version. YouTrack 2026.1.1+ recommended for full functionality (deep-link URLs, browser back/forward navigation).

1. Install the app

Upload the app ZIP to your YouTrack instance via Admin → Apps.

2. Enable for a project

Go to Project Settings → Portal tab. Click Enable Service Center for the project.

3. Create forms

In the same Project Settings → Portal tab, add categories and create forms. Configure fields, mappings, and templates. Publish the form to make it visible in the portal.

4. Open Service Center

Service Portal appears in the YouTrack sidebar menu. Click it to see the catalog of published forms. Users can fill and submit forms to create YouTrack issues.

5. Settings

Administration → Apps → Adv.ServiceCenter → Settings — configure logging level (no_log, minimal, debug).

6. Documentation

See README.md for an overview and HOW_TO_USE.md for the practical guide.
